## üìÇ Page Overviews

- **Home (`index.html`)**  
  The central hub of the portfolio. It highlights my academic journey at Northeastern, professional skills (rendered via dynamic JS charts), and contact information.

- **Projects (`project.html`)**  
  A dedicated showcase of my work. It features an automated gallery that renders project metadata from a structured local source, demonstrating modular component building.

- **Cat World (`cats.html`)**  
  An interactive experimentation page. Includes a virtual petting counter that tracks state and a randomized fact generator to provide an engaging user experience.

- **Coffee Corner (`coffee.html`)**  
  A premium, theme-specific page dedicated to the art of brewing. It includes a real-time coffee extraction calculator powered by custom JavaScript logic.
